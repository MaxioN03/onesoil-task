# OneSoil Test Task

### To run project:
1. ```git clone https://github.com/MaxioN03/onesoil-task.git```
2. ```cd onesoil-task```
3. ```npm install```
4. ```npm start```

### To run StoryBook:
* ```npm run storybook```
